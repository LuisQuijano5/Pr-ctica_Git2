## Descripción del proyecto
Este repositorio se desarrolló como práctica para comprender y aplicar un flujo de trabajo colaborativo usando **Git** y **GitHub**.  
Se emplearon ramas, commits descriptivos, pull requests, resolución de conflictos y documentación del flujo de trabajo, simulando un entorno real de desarrollo colaborativo.

---

## Objetivo
Familiarizarnos con los comandos y procesos más importantes de Git/GitHub en un equipo, incluyendo:
- Creación de repositorio remoto y configuración de colaboradores.
- Trabajo en ramas separadas para cada integrante.
- Realización de commits claros y significativos.
- Uso de `git log`, `git pull`, `git revert`, `merge` y `pull request`.
- Resolución de conflictos de manera adecuada.
- Documentación del flujo de trabajo y aprendizajes obtenidos.
